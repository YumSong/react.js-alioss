# react.js-alioss
前台实现阿里云上传图片

# 组件下载

npm install  uploadimage_alioss  --save-dev

# example_app 运行
npm install 
npm run start



# uploadimage_alioss  插件说明
 

# 阿里云插件引用方式 

`bash
1. <script src="https://gosspublic.alicdn.com/aliyun-oss-sdk.min.js"></script>
2. require('ali-oss/dist/aliyun-oss-sdk.js')   //不直接引用的原因  不支持es6

`
